SwipyCell
Easy to use UITableViewCell implementing swiping to trigger actions
state is a representation of the trigger point for that swipe trigger,             Example: .state(0, .left) mode is either .exit or .toggle, .exit finishes the slide, .toggle bounces the cell back swipeView is for displaying icons, etc. below the swiping cell swipeColor is the background color for the swipeView
cell.addSwipeTrigger(forState: SwipyCellState,                      withMode: SwipyCellMode,                      swipeView: UIView,                      swipeColor: UIColor,                      completion: SwipyCellTriggerBlock?                     )
To setup a new trigger for an existing SwipyCell object, call
